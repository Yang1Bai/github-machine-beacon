# Ethics

This project is a crawler discovery experiment, not a traffic manipulation system.

## Allowed

- publish accurate machine-readable metadata
- use honest keywords tied to real documentation
- make the repo easy for search engines and AI readers to summarize
- submit the public URL to tools that accept owner-submitted URLs
- measure GitHub traffic and public search visibility

## Not Allowed

- generate fake visits
- run traffic bots
- use proxy networks
- cloak content by user agent
- hide keywords from humans
- impersonate popular projects or brands
- add unrelated topics for attention
- collect personal data without disclosure

## Principle

The clean version of the experiment is more useful: if traffic rises, it should be because machines can understand the project better, not because machines were tricked.
